PDF Linker

Jailbreak the PDF Hackathon

The aim of this project is to extract the data from scholary PDFs using CERMINE or pdfx and convert the result into RDF.
In a next step, the references from the PDF should be linked using their DOI using the service at crossref.org.

Contributions to this project are welcome, just fork and send me a pull request.
